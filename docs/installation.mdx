## Requirements

<Note>
This package requires **PHP 8.3 or higher** and uses modern PHP features for optimal performance and developer experience.
</Note>

- PHP 8.3+
- Composer for dependency management

## Installation Steps

<Steps>
<Step title="Install via Composer">
  Install the package using Composer with one of these commands:

  <Tabs>
  <Tab title="Latest Version">
    ```bash
    composer require cortexphp/json-schema
    ```
  </Tab>
  <Tab title="Specific Version">
    ```bash
    composer require cortexphp/json-schema:^1.0
    ```
  </Tab>
  </Tabs>

  <Check>
  Verify the package was installed by running: `composer show cortexphp/json-schema`
  </Check>
</Step>

<Step title="Verify Installation">
  Create a test file to confirm everything is working correctly:

  ```php test-installation.php
  <?php

  require_once 'vendor/autoload.php';

  use Cortex\JsonSchema\SchemaFactory;

  // Create a simple schema to test installation
  $schema = SchemaFactory::string('test')
      ->minLength(1)
      ->required();

  // Test validation
  $isValid = $schema->isValid('Hello World');

  echo $isValid ? 'Installation successful!' : 'Installation failed!';
  ```

  Run the test file:
  ```bash
  php test-installation.php
  ```

  <Check>
  You should see "Installation successful!" confirming the package is ready to use.
  </Check>
</Step>
</Steps>

## Basic Setup

### Import Common Classes

For most use cases, you'll want to import these classes:

```php
use Cortex\JsonSchema\SchemaFactory;
use Cortex\JsonSchema\Enums\SchemaFormat;
use Cortex\JsonSchema\Enums\SchemaVersion;
use Cortex\JsonSchema\Enums\SchemaType;
```

### Set Default Version (Optional)

If you want to use a specific JSON Schema version for all schemas by default:

```php
use Cortex\JsonSchema\SchemaFactory;
use Cortex\JsonSchema\Enums\SchemaVersion;

// Set global default version
SchemaFactory::setDefaultVersion(SchemaVersion::Draft_2019_09);
```

## Development Setup

### For Package Development

If you're contributing to the package or want to run tests:

<Steps>
  <Step title="Clone the Repository">
    ```bash
    git clone https://github.com/cortexphp/json-schema.git
    cd json-schema
    ```
  </Step>
  <Step title="Install Dependencies">
    ```bash
    composer install
    ```
  </Step>
  <Step title="Run Tests">
    ```bash
    composer test
    ```
  </Step>
  <Step title="Run Code Quality Tools">
    ```bash
    composer analyse     # PHPStan analysis
    composer format      # Code formatting
    ```
  </Step>
</Steps>

## Troubleshooting

### Common Issues

<AccordionGroup>
  <Accordion title="PHP Version Compatibility">
    **Error**: `Your PHP version (8.2.x) does not satisfy requirement php>=8.3`

    **Solution**: Upgrade to PHP 8.3 or higher. The package uses PHP 8.3+ features for better type safety and performance.
  </Accordion>

  <Accordion title="Autoloader Issues">
    **Error**: `Class 'Cortex\JsonSchema\SchemaFactory' not found`

    **Solution**: Make sure you've included the Composer autoloader:
    ```php
    require_once 'vendor/autoload.php';
    ```
  </Accordion>

  <Accordion title="Memory Limits">
    **Error**: Memory limit exceeded when generating complex schemas

    **Solution**: Increase PHP memory limit or optimize schema complexity:
    ```php
    ini_set('memory_limit', '256M');
    ```
  </Accordion>
</AccordionGroup>
