## Requirements

- PHP 8.3+
- Composer for dependency management

## Installation Steps

<Steps>
<Step title="Install via Composer">
  ```bash
    composer require cortexphp/json-schema
  ```
</Step>

<Step title="Set Default Version (Optional)">
  The package uses **Draft 2020-12** as the default version. If you need to use a different version for all schemas:

  ```php
  use Cortex\JsonSchema\Schema;
  use Cortex\JsonSchema\Enums\SchemaVersion;

  // Override default version (if needed)
  Schema::setDefaultVersion(SchemaVersion::Draft_06); // For maximum compatibility
  // or
  Schema::setDefaultVersion(SchemaVersion::Draft_2019_09); // For balanced features
  ```
</Step>
</Steps>

## Development Setup

### For Package Development

If you're contributing to the package or want to run tests:

<Steps>
  <Step title="Clone the Repository">
    ```bash
    git clone https://github.com/cortexphp/json-schema.git
    cd json-schema
    ```
  </Step>
  <Step title="Install Dependencies">
    ```bash
    composer install
    ```
  </Step>
  <Step title="Run Tests">
    ```bash
    composer test
    ```
  </Step>
  <Step title="Run Tests and Code Quality Tools">
    ```bash
    composer check
    ```
  </Step>
</Steps>
